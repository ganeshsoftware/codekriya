C# Programming

C# PL is derived from C and C++ (syntax, semantics etc). It is the PL recommended by MS to develop Web, Desktop, Mobile and Cloud based applications targetting the .NET Framework. Learn the basics (the foundation and the most time consuming) aspects of the C# programming language and the .NET Framework.

Object Oriented Programming

C# is a Object Oriented Programming (OOP) language. This means a programmer can create class templates and make instances of these templates. On top of that, the programmer can leverage the Encapsulation, Inheritance, Polymorphism and Abstraction features. This section will deal with how we implement these things in C#.

C# Collections

C#(and other languages of the .NET Framework), in addition to the basic data types has the provision to create Collections. Arrays, Arraylist, Stack<T>, Queue<T>, Hashtables, Dictionary<TKey, TValue> are all examples of Collections that ara available as classes in the System.Collections.Generic namespace.
